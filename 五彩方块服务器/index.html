前言

在大约八年前的春节，仓鼠拿着自己仅有的一百块钱压岁钱，在淘宝网上租了一个服务器，那就是最初的五彩方块。
第一次独立开服还算顺利，还记得当年在网吧打开Minecraft看到自己服务器在线玩家首次达到了20位，连作为服主的自己都挤不进去服务器了。非常开心，激动地把坐在旁边的华仔暴揍了一顿。
自那以后便一直在开着服务器了，期间也做了许多付费内容，并一直都依靠这个来维持服务器的开支。
在开Minecraft的这八年时间里，也见识过许多事情，结识了许多朋友。而最重要的是，有那么一些玩家始终和我在一起，每一个周目都有他们的影子。
曾经在五彩方块中游戏的玩家们，有一部分组建了自己的玩家团体，建立了它们自己的QQ群。时至今日，它们的QQ群已经拥有数百名群员，而游戏内容早已不再是单纯的Minecraft。他们玩彩虹六号、CSGO、GTAV、方舟：生存进化，但他们都是因为我而相遇。
也许你会觉得我在矫情，但就是这样的事情给我带来了成就感。
当年才上初二，我想给自己的服务器增添更多乐趣，于是决定去学习Java语言编写插件，只可惜当我学完这些内容、真正可以编写代码时，我已经进入了高中，无力再建设Minecraft服务器了。但我仍然是幸运的，因为Minecraft，我找到了自己的兴趣所在，上大学时选择了软件专业。
而现在我已经大学毕业，找到了一份自己满意的工作（也是写Java），有了充足的资金，也结识了许多好友（为我免费提供服务器和技术支持），更重要的是，我拥有了比以往任何时候都要更丰富的技术知识和经验。
所以，我花了许多时间，想要再一次开启五彩方块。

服务器简介

五彩方块是一个以高版本生存为核心的Minecraft服务器，我们希望玩家能够在这里体验到生存游戏的独特趣味。
在五彩方块服务器里没有死亡保护：死亡将会让你丢失身上所有的装备和经验值。
五彩方块给玩家提供极高的自由度，除了少量破坏性极大（让服务器崩溃或导致服务器严重卡顿）的漏洞以外，我们几乎不禁止其他任何行为。这意味着你可以随意烧杀抢掠其他玩家的财产，而不会受到服务器官方的制裁（但也许会有玩家团队惩罚你）。
但是在Minecraft中生存始终是孤独的，这也是许多玩家选择在服务器中游玩生存模式的主要原因。所以，为了应对生存游戏的孤独感，以及进一步增强玩家之间的交流，我们建立了一个联盟系统，意在把游戏中互相孤立的玩家们联结起来。
之后我们会增添许多其他游戏元素，不断丰富五彩方块服务器的游戏玩法。

服务器90%原创插件，其中部分插件已开源，开源插件可以在 github 上找到。

服务器内容

联盟系统


五彩方块服务器的联盟系统允许玩家加入服务器内已有的联盟，也可以自己创建联盟并邀请其他玩家加入。
联盟的首席执行官可以在地图中创建一个城市区域并扩建这个区域，所有扩建的区域都可以在 卫星地图 上实时显示。
当城市创建完成后，联盟的首席执行官可以为城市设定权限来保护自己的城市免受外人的侵害。
这一个系统类似于大多数服务器都拥有的领地插件，不同之处在于领地是以玩家为单位的保护区域，而联盟则代表着一个玩家群体。
这也就意味着在五彩方块，如果你想要保护自己的个人资产，那么你就不得不加入某一个联盟，并与联盟中的其他所有人共享资源，而这也是我打造联盟系统的初衷：将所有玩家联结起来。
但是仍然有一种例外的情况：如果你的实力足够强大，亦或者不相信除自己以外的任何人，那么你也可以选择独自一人（付出极大的代价）创建联盟。

每月备份

部分不开源的原创插件也可提供给超过两个月以上的玩家。
